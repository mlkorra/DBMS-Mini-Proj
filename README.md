# Car Booking System

## Done as a part of Database Management Systems Course Projects.

### Functionalities

**Users**

  * SignUp/Login for the Users
  * Booking Cabs
  * Viewing Previous Bookings
  * Cancel Bookings
  * Update User Profile
  
**Admin**
  * Add/Remove New Cars,Locations.
  * View all the registered users.
  * Update Admin Profile
